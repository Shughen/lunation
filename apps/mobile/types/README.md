# /types

Interfaces et types TypeScript partagés.

## Fichiers

- `api.ts` - Types pour les réponses API (LunarReturn, NatalChart, etc.)
- `stores.ts` - Types pour les états Zustand
- `storage.ts` - Clés AsyncStorage centralisées + types

## Règles

- Tous les types doivent être exportés
- Préférer `interface` à `type` pour les objets
- Documenter les types complexes avec JSDoc
